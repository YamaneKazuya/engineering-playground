
# laravelフレームワークの遊び場

## 概要
docker環境でlaravelフレームワークの基本構築方法を作る
また、活用方法をまとめておく場所

## Setup

### 1. Clone
git clone

### 2. laravelプロジェクトを作成
composer create-project laravel/laravel .

## 拡張機能
### [laravel boost](https://laravel.com/docs/12.x/boost)

バッググラウンドで起動する<br>
php artisan boost:mcp > /dev/null 2>&1 &
